## Installation

In your command line, run the following commands:

```
$ git clone https://github.com/HighlandsCollege/diogo-maths-app.git
$ cd maths-app-diogo

# To check Flutter environment
$ flutter doctor

# After connecting a device run:
$ flutter run
```
